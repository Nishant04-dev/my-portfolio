# 🚀 Nishant Chauhan | Cinematic Portfolio

A high-performance, premium "Developer & Entrepreneur" showcase built with React, Vite, and Framer Motion. This portfolio features a unique **Premium Crimson** aesthetic and is optimized for peak performance across all devices.

## ✨ Features

- **🔴 Premium Crimson Theme**: A deep, sophisticated dark mode with ruby-crimson accents and glassmorphism.
- **🚀 Ultra-Optimized Performance**: 
  - Isolated high-frequency re-renders (Typing effect).
  - Optimized $O(N^2)$ Canvas calculations.
  - Zero `backdrop-blur` for maximum framerate on mobile.
- **📱 Fully Responsive**: Seamless experience from desktop 5K monitors to high-end smartphones.
- **🎨 Advanced Animations**: Silky smooth parallax, scroll-reveal effects, and interactive particle backgrounds.
- **🔥 Developer-First Aesthetic**: Clean typography (Bebas Neue & Inter) and logical layout.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Performance Optimizations

Unlike traditional heavy portfolios, this showcase is built for speed:
1. **Canvas Debouncing**: Particle connection loops were removed to prevent CPU bottlenecks.
2. **GPU Acceleration**: Leverage `will-change` properties and motion values for 60FPS scrolling.
3. **Lazy Loading**: All secondary sections are code-split using `React.lazy` to minimize initial bundle size.
4. **Isolated States**: High-frequency updates are encapsulated to prevent full-page reconciliation.

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nishant04-dev/cinematic-showcase.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
